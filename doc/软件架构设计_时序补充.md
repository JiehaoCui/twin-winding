```
| M  | T  | W  | S  | Sh | C  | St | Pr | Fa |    |
| o  | e  | i  | t  | a  | h  | a  | u  |    |    |
| t  | n  | n  | a  | p  | a  | t  | d  | l  |    |
| o  | s  | d  | r  | i  | n  |    | u  | t  |    |

|    | o  | n  | /  | g  | e  |    | t  |    |    |

+----+----+----+----+----+----+----+----+----+
```

- Motor (M): 电机控制（10ms）
- Tension (T): 张力闭环（10ms）
- Winding (W): 卷绕控制（10ms）
- Start/Stop (St): 启停控制（10ms）
- Shaping (Sh): 成型控制（10ms）
- Changeover (C): 换卷控制（10ms）
- StateMachine (St): 状态机（10ms）
- Production (Pr): 产量统计（10ms）
- Fault (Fa): 故障处理（10ms）

**1ms周期任务**（高优先级，独立运行）:
```
+---+---+---+---+---+---+---+---+
```

- Traverse (Tr): 排线控制（1ms，最高优先级）

**50ms周期任务**:

- Button (Bu): 按键扫描（50ms）

### 4.2 换卷时序

```
满卷检测 → 换卷触发 → 速度匹配 → 步进电机换卷 → 张力切换 → 步进复位 → 完成
  (10ms)    (10ms)      (500ms~1s)   (200ms)     (10ms)     (200ms)    (10ms)
```

## 五、初始化流程

```
上电
  ↓
硬件初始化（GPIO/TIM/UART/ADC/I2C）
  ↓
驱动层初始化（电机/通信/IO/ADC）
  ↓
业务层初始化（参数/统计/运动规划）
  ↓
应用层初始化（所有模块Init）
  ↓
创建FreeRTOS任务
  ↓
启动调度器
  ↓
进入状态机：STATE_INIT
  ↓
检测3个驱动器通信（内部485）
  ↓
排线电机回零（堵转）
  ↓
等待工位选择（车头PLC下发）
  ↓
初始化完成 → 进入STATE_IDLE
```

### 7.2 新增/拆分模块说明

1. **按键扫描模块**: 按键面板去抖和事件检测（HTR3335）
2. **排线控制模块**: 排线电机往复运动控制（1ms，高优先级），原"排线控制"拆分后保留往复控制功能
3. **成型控制模块**: 排线动程计算、收边角度处理、排线基准转速计算（10ms），从原"排线控制"中拆分
4. **产量统计模块**: 米长统计和直径计算（10ms），原名"统计计算"改为"产量统计"
5. **参数管理模块**: 集中管理所有PLC下发的参数
6. **统计计算模块**: 产量、运行时间、满卷次数统计
